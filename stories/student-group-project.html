FICTIONAL DIALOGUE DRAMA - SCHOOL LESSONS

Smug Group Leader Took Credit For The Whole Project. The File History Showed Who Built Every Slide.

He wanted the applause and the grade. The shared document had been keeping score the entire time.

Published July 1, 2026 - 12 min read - Fictional dialogue-format drama

Fiction notice: This is a fictional dialogue drama. All characters, messages, workplaces, schools, businesses, places, and events are invented for entertainment.

1. Project Week

Biology Project Thread: Presentation is Friday. Everyone please update your assigned slides.

Quiet Scholarship Student: I finished the research summary and graph draft.

Smug Group Leader: Good. Send it to me so I can make it presentable.

Overworked Part-Time Student: I added the citation list.

Smug Group Leader: I will clean that too.

Public-Shaming Teacher: Reminder: individual contribution records may be reviewed.

Smug Group Leader: No worries. I am managing the group.

Quiet Scholarship Student: I can present the data section.

Smug Group Leader: Let's not risk it. I will present.

2. Work Disappears

File History: Quiet Scholarship Student created Slides 4-9 Monday 8:11 PM.

File History: Overworked Part-Time Student added sources Monday 9:03 PM.

Smug Group Leader: I renamed the file so it looks professional.

Quiet Scholarship Student: Where did my name go from the title slide?

Smug Group Leader: Too many names look messy.

Overworked Part-Time Student: It is a group project.

Smug Group Leader: Exactly. The group does not need a billboard.

Quiet Scholarship Student: The rubric asks for names.

Smug Group Leader: I know the teacher better than you do.

File History: Title slide edited by Smug Group Leader Tuesday 12:02 AM.

3. Stay Quiet

Private Thread - Smug Group Leader: Do not make this weird tomorrow.

Private Thread - Quiet Scholarship Student: Make what weird?

Private Thread - Smug Group Leader: The credit thing.

Private Thread - Quiet Scholarship Student: I did the data section.

Private Thread - Smug Group Leader: And I am presenting it. That helps you.

Private Thread - Quiet Scholarship Student: I would like my contribution shown.

Private Thread - Smug Group Leader: You should be grateful. People freeze when they present.

Private Thread - Quiet Scholarship Student: I do not.

Private Thread - Smug Group Leader: Do not challenge me in front of the class.

Private Thread - Quiet Scholarship Student: Saved.

4. The Document Remembers

File History: Data chart created by Quiet Scholarship Student Tuesday 6:42 PM.

File History: Slide 10 deleted by Smug Group Leader Wednesday 12:14 AM.

Overworked Part-Time Student: Why was my sources slide moved to backup?

Smug Group Leader: It broke the flow.

Quiet Scholarship Student: The teacher requires visible citations.

Smug Group Leader: I will mention sources verbally.

Public-Shaming Teacher: Citations must appear in the presentation file.

Overworked Part-Time Student: Teacher just posted that.

Smug Group Leader: Fine. I will add them later.

File History: No citation edits by Smug Group Leader detected.

5. Presentation Day

Class Presentation Thread: Group 4, you are next.

Smug Group Leader: I handled most of the research and design, so I will walk everyone through it.

Quiet Scholarship Student: Most?

Smug Group Leader: Please do not interrupt.

Public-Shaming Teacher: Let him begin.

Smug Group Leader: Our data shows a clear pattern. I built this model after reviewing the raw results.

Quiet Scholarship Student: I built the model.

Smug Group Leader: We discussed this.

Classmate: Awkward.

Public-Shaming Teacher: Save disputes for after.

Smug Group Leader: Thank you. As I was saying, my chart shows...

Presentation Clicker: Slide 7 skipped.

6. The Lie Gets Bigger

Smug Group Leader: The skipped slide was not important.

Quiet Scholarship Student: It was the method slide.

Smug Group Leader: I said not important.

Public-Shaming Teacher: Continue.

Smug Group Leader: I spent three nights cleaning everyone's messy work.

Overworked Part-Time Student: That is not true.

Smug Group Leader: Some people contributed emotionally.

Classmate: Ouch.

Quiet Scholarship Student: Please open the file history.

Smug Group Leader: This is desperate.

Public-Shaming Teacher: We will review after the presentation.

Smug Group Leader: Great. The grade will speak for itself.

7. Teacher Review

Public-Shaming Teacher: Group 4, stay after class.

Smug Group Leader: I hope this is quick.

Quiet Scholarship Student: It will be if we open version history.

Overworked Part-Time Student: I also have screenshots of the source slide before it was moved.

Public-Shaming Teacher: Open the shared file.

File History: Version history opened by Public-Shaming Teacher.

Smug Group Leader: Version history can be misleading.

Public-Shaming Teacher: It lists edits by account.

Quiet Scholarship Student: Yes.

Smug Group Leader: Fine. Look.

8. Every Slide Has A Name

File History: Slides 4-9 created by Quiet Scholarship Student.

File History: Graph data imported by Quiet Scholarship Student.

File History: Citation slide created by Overworked Part-Time Student.

File History: Title names removed by Smug Group Leader.

File History: Method slide skipped during presentation by Smug Group Leader.

Public-Shaming Teacher: Why did you say you built the model?

Smug Group Leader: I supervised it.

Quiet Scholarship Student: You deleted my name.

Overworked Part-Time Student: And hid my sources slide.

Public-Shaming Teacher: That is not supervision. That is misrepresentation.

Smug Group Leader: I was trying to make us look unified.

Public-Shaming Teacher: You made yourself look dishonest.

9. Grade Split

Public-Shaming Teacher: The project grade will be evaluated individually.

Smug Group Leader: That is unfair.

Public-Shaming Teacher: It is exactly what the rubric says.

Quiet Scholarship Student: Can I present the skipped method slide?

Public-Shaming Teacher: Yes. Tomorrow, before grades are finalized.

Overworked Part-Time Student: Can my citation slide be restored?

Public-Shaming Teacher: It already is.

File History: Citation slide restored by Public-Shaming Teacher.

Smug Group Leader: So I get punished for leading?

Public-Shaming Teacher: You are being graded for your actual contribution.

Classroom System: Individual review notes added.

Quiet Scholarship Student: Thank you.

10. Quiet Credit

Private Thread - Public-Shaming Teacher: I owe you an apology for stopping you too quickly.

Private Thread - Quiet Scholarship Student: Thank you for saying that.

Private Thread - Public-Shaming Teacher: Your method explanation was strong.

Private Thread - Quiet Scholarship Student: I wanted the work to be seen.

Private Thread - Public-Shaming Teacher: It is now.

Biology Project Thread: Final grades posted.

Overworked Part-Time Student: I got full source credit.

Quiet Scholarship Student: I got the research credit.

Smug Group Leader: This group was impossible.

Overworked Part-Time Student: The file history seemed fine with us.

Quiet Scholarship Student: Good luck on finals.

File History: Project archived.
