How Original Archive Records and Creator Scripts Stay Separate

Kyunolab keeps Archive records and Creator Library scripts connected but separate. This guide explains why that boundary matters for readers and creators.

Archive is for reading

The Mystery Archive presents the record as a reading experience. It focuses on story, source status, meaning, atmosphere, and related archive paths.

It does not need to behave like a production checklist.

Creator Library is for making

The Creator Library turns selected archive material into production-ready assets. It gives narration, prompts, music direction, editing guidance, and optional advanced notes.

Its purpose is not just to explain the story. Its purpose is to help someone make a finished video.

Connections should be clear, not blended

A script page may link back to the original record, but it should not replace that record. The reading page and production page should each keep their own role.

That separation helps the site stay understandable as both the Archive and the Creator Library grow.

The boundary protects trust

Legends, folklore, mysteries, and invented narrative treatments can easily blur together. Clear page roles help prevent that confusion.

The Archive preserves the record. The Library adapts the record for production.
