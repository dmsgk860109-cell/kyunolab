How Creator Library Categories Organize Script Packages

Creator Library categories help creators find production-ready script packages. They are connected to the archive, but they are not the same as archive reading categories.

Categories act like production shelves

In the Creator Library, a category is a shelf of script packages. It helps the creator find topics by the kind of video they want to make.

Urban Legends, Internet Folklore, Myths, and Mythic Objects may all need different pacing, visuals, and tone.

Category does not change the template

Every category still uses the same production structure. Story comes first, then Toolkit, Workflow, Long-form Creator, Short-form Creator, and reference material.

That consistency lets creators move between topics without learning a new page format each time.

The category sets expectations

A category helps the creator anticipate the kind of assets they may need. A haunted place may lean on atmosphere, while a mythic object may need a clear central prop.

The Scene details still make the final decisions, but the category gives the first production direction.

Archive and Library stay separate

Archive categories organize reading records. Library categories organize creator materials.

The two systems can point to each other, but they serve different jobs.
