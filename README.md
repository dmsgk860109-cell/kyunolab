# Kyunolab Mystery Archive

Static Cloudflare Pages site for kyunolab.com.

The site is a quiet archive for urban legends, internet folklore, classic myths, strange places, mythic creatures, and mysterious stories from around the world.

## Deploy

Cloudflare Pages is connected to the GitHub repository. Pushes to main deploy automatically.

## Build

Run `npm run build` before deploy. The build generates tag archive pages from article metadata and updates `sitemap.xml`.
